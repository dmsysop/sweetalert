Configuration

In configuration files we have defined default values and also supported values for our sweetalert package , default values will be used in which case when you forget to define argument in alert function except title , because title is required to show alert popup.

Change the default array value to make it available by default without defining it on a function , you can edit at at config/sweetalert.php


return [

    'default' => [
        'title'             => '',
        'text'              => '',
        'timer'             => '',
        'width'             => '',
        'heightAuto'        => '',
        'padding'           => '',
        'animation'         => '',
        'showConfirmButton' => false,
        'showCloseButton'   => true,
    ],

    /**
    * These are supported arguments which can be used in alert function.

    'supported' => [
